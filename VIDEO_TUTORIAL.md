# 视频教程脚本

> 适用于录屏讲解，5分钟完成基础安装

---

## 片段一：开场（30秒）

```
大家好，今天教大家5分钟在Windows上安装Hermes Agent。
这是一个会自我进化的AI智能体，安装非常简单，
学完之后你可以在飞书或微信里跟它对话，让它帮你完成各种任务。
```

---

## 片段二：WSL2安装（2分钟）

```
首先我们需要启用WSL2。
按Win键，搜索PowerShell，右键以管理员身份运行。
复制粘贴这段命令...
dism.exe /online /enable-feature...
```

**需要录屏的内容：**
1. 打开 PowerShell（管理员）
2. 运行 WSL2 启用命令
3. 重启提示
4. 运行 `wsl --install` 安装 Ubuntu

---

## 片段三：Hermes安装（1.5分钟）

```
现在打开Ubuntu终端，粘贴安装命令...
curl -fsSL https://...
等待几分钟，看到这个提示就成功了。
验证一下，输入 hermes --version
```

## 录制小技巧

1. **提前测试一遍** 再开始录
2. **语速放慢**，边操作边讲解
3. **鼠标指向** 要点击的地方停留1-2秒
4. **关键步骤放大** 鼠标操作区域
5. **准备背景音乐** 轻微的背景音更有质感
